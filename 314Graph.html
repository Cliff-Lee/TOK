Tree checker

What to notice

A subgraph uses some of the vertices and some of the edges of the original graph.

A tree is a connected graph with no cycles.

For an undirected tree with n vertices, the number of edges is n − 1.

Helpful checks

Does the selected graph stay connected?

Does it contain any cycle?

Does it have exactly one fewer edge than vertices?

Student Activities

Activity 1: Create a graph with 5 vertices. Identify two adjacent vertices and two adjacent edges.
Activity 2: Make a complete graph on 4 vertices. What is the degree of each vertex?
Activity 3: Switch on Directed. Create a graph where one vertex has in-degree 2 and out-degree 1.
Activity 4: Create a weighted graph to model roads between towns. Explain what the weights mean.
Activity 5: Select part of a graph to form a subgraph. Explain why it is a subgraph.
Activity 6: Build a tree with 6 vertices. What pattern do you notice between the number of vertices and edges?
Activity 7: Try to build a graph that is connected but not complete. Then build one that is complete but not a tree.
