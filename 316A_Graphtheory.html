AHL 3.16A — Graph Structure Explorer

Build graphs, classify routes, test Eulerian and Hamiltonian properties, and explore Kruskal’s and Prim’s minimum spanning tree algorithms.

Main ideas

Route types

Distinguish between walk, trail, path, circuit, and cycle by checking what repeats.

Eulerian

Use connectedness and odd/even degree to decide whether an Eulerian trail or circuit exists.

Hamiltonian

Investigate paths and cycles that visit every vertex exactly once.

Spanning trees

Understand what a tree is and how a minimum spanning tree connects all vertices with smallest total weight.

Big distinction: Eulerian questions are about using every edge. Hamiltonian questions are about visiting every vertex.

What students should be able to do

Tell whether a sequence is a valid walk, trail, path, circuit, or cycle.

Determine whether an Eulerian trail or circuit exists and explain why.

Search for Hamiltonian paths and cycles.

Build a minimum spanning tree using Kruskal or Prim.

Use a weighted adjacency table to support Prim’s algorithm.

What the applet checks

Walk: every consecutive pair must be joined by an edge.

Trail: no repeated edge.

Path: no repeated vertex.

Circuit: closed trail.

Cycle: closed path.

Visual feedback

The route’s edges and vertices are highlighted on the graph so students can see exactly what is being checked.

When MST makes sense

MST is appropriate for a graph that is weighted, undirected, and connected.

The applet explains when one of these conditions fails.

Algorithm summaries

Kruskal: choose the lightest available edge that does not create a cycle.

Prim: start at one vertex and repeatedly add the lightest edge that joins the current tree to a new vertex.

Matrix support for Prim

Use the weighted adjacency table in the Graph Builder tab to follow Prim in a matrix-style way.

Student Challenges

Challenge 1: Make a sequence that is a trail but not a path.
Challenge 2: Build a graph with exactly two odd vertices and predict the Eulerian result.
Challenge 3: Create a graph with a Hamiltonian cycle but no Eulerian circuit.
