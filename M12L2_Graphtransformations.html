Explore Quadratic Transformations

Discover how the parent graph y = x² changes into y = a(x - h)² + k. Move the sliders, compare graphs, describe the transformation, and use the vertex form to write equations from graphs.

Focus
Vertex form, graph features, and transformation language.

Controls

a value 1
a changes the width and direction. If |a| > 1, the graph becomes narrower. If 0 < |a| < 1, it becomes wider. If a < 0, it reflects in the x-axis.
h value 0
h moves the graph left or right. In (x - h), a positive h moves the graph to the right.
k value 0
k moves the graph up or down. Positive k moves the graph up.
