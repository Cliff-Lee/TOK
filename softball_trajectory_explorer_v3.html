Softball Trajectory Explorer

An interdisciplinary Mathematics + PE applet for experimenting with softball flight paths. The side view is modeled with a quadratic, while the front and top views show how the same path looks from different viewpoints for tactics, catching, and coaching discussion.

If the applet does not display correctly when opened directly from disk, try opening it in Chrome with a small local server instead of double-clicking the file.

Model controls

How to use this
  1. Create or choose a trajectory.
  2. Study the same path from the side, front, and top views.
  3. Use the equation, vertex, intercept, range, and hang time to describe the motion.
  4. Then connect the mathematics to PE decisions such as catching, field placement, shot choice, and consistency.
